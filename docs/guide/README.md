## Installation

👉 Use the [install](./install/README.md) guide to get set up.

## GUI

The simplest way to experience Lemonade is to explore the built-in GUI. The GUI helps you obtain and manage models, try them out, and explore the local AI ecosystem.

=== "Windows"

    👉 Click the 🍋 tray icon and select Open Lemonade App

=== "macOS"

    👉 Run the Lemonade App from your apps folder.

=== "Linux"

    👉 Open http://localhost:13305 in your browser.

> Tip: the web ui is available on all platforms at http://localhost:13305
