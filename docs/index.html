Unified API

One local endpoint for chat, vision, images, audio, and embeddings.

OpenAI-compatible request shapes with local execution. Point your tools at Lemonade and swap models without rewriting integrations.

Quickstart

Drop this into your existing OpenAI-compatible clients. Default local base URL:

curl http://localhost:8000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d "{\"model\":\"qwen3\",\"messages\":[{\"role\":\"user\",\"content\":\"Summarize this repo in 5 bullets\"}]}"

Developer ergonomics

  • Single endpoint family for all modalities
  • Cloud-compatible payload formats
  • Run mixed-model workloads on one local server
  • Switch engines based on available GPU or NPU
