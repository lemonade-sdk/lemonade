    Applies to capabilities

    Presets fit any model that exposes one of these capabilities. Capability labels (chat, embedding, image, transcription, …) come from the model's registered tools — they don't depend on which backend (llamacpp, flm, ryzenai-llm, …) is serving it.

    Advanced flags (*_args)

    Passed verbatim to whatever backend serves the model at load time. Keys map to the backend's option set (e.g. llamacpp_args, flm_args).
